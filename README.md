### private work

Some projects are private for client or confidentiality reasons. Brief descriptions below.

**AI fitness dashboard** (private, deployed on Vercel)
Full-stack dashboard built for personal training use. Tracks workouts, running sessions, and body metrics over time. Connects to an AI layer that reads the data and gives weekly feedback: recovery suggestions, load warnings, training balance. Built with Next.js, TypeScript, Supabase, and a Recharts visualization layer. The AI component uses structured prompting to keep outputs consistent and actionable rather than generic.

**Hybridium OS** (private, deployed on Vercel)
A browser-based simulated operating system interface. Built as an experiment in complex UI state management: multiple draggable windows, a working file system in memory, app-to-app communication, and a custom terminal. TypeScript and React throughout. The point was to push component architecture and state patterns beyond what a standard app requires.

**Client yoga website** (private, live in production)
Front-end website built for a real client. Booking flow, class schedule, mobile-first responsive design. JavaScript, deployed and actively used.
